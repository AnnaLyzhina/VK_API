# Vk

Учебный дипломный проект по курсу «Python-разработчик с нуля».

Проект представляет собой VK-бота для поиска людей во ВКонтакте по параметрам:

- возраст;
- пол;
- город.

Бот показывает имя, фамилию, ссылку на профиль и 3 самые популярные фотографии кандидата по количеству лайков.

Проект специально переписан без SQLAlchemy. Для работы с PostgreSQL используется только `psycopg2-binary`, потому что так проект проще устанавливается на Windows и учебных компьютерах.

---

## Что умеет бот

- искать кандидатов через VK API;
- получать 3 популярные фотографии из профиля;
- отправлять карточку кандидата в чат VK;
- переходить к следующему кандидату;
- сохранять кандидата в избранное;
- показывать список избранных;
- добавлять кандидата в чёрный список;
- сохранять данные в PostgreSQL;
- работать через кнопки VK.

## Установка проекта

### 1. Распаковать архив

Например:

```bash
C:\Projects\Vk
```

### 2. Открыть папку проекта в VS Code

Открывать нужно именно папку `Vk`, где лежит файл `main.py`.

### 3. Создать виртуальное окружение

```bash
python -m venv venv
```

### 4. Активировать виртуальное окружение

Windows PowerShell:

```bash
venv\Scripts\activate
```

Если PowerShell запрещает запуск:

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

Потом снова:

```bash
venv\Scripts\activate
```

### 5. Обновить pip

```bash
python -m pip install --upgrade pip
```

### 6. Установить зависимости

```bash
pip install -r requirements.txt
```

## Настройка .env

Скопируйте файл:

```text
.env.example
```

и переименуйте копию в:

```text
.env
```

Заполните значения:

```env
TOKEN=токен_группы_vk
USER_TOKEN=токен_пользователя_vk
GROUP_ID=id_группы
DB_NAME=vk
DB_USER=postgres
DB_PASSWORD=пароль_postgresql
DB_HOST=localhost
DB_PORT=5432
```

### TOKEN

Это токен сообщества VK.

Получается в настройках группы:

```text
Управление -> Работа с API -> Ключи доступа
```

Нужные права:

```text
messages, photos, users, groups
```

### USER_TOKEN

Это токен пользователя VK.

Он нужен для поиска людей и получения фотографий.

### GROUP_ID

Если ссылка на группу такая:

```text
https://vk.com/public123456789
```

то в `.env` нужно указать:

```env
GROUP_ID=123456789
```

Без слова `public`.

## Команды бота

- `Привет` — начать поиск;
- `Следующий` — показать следующего кандидата;
- `В избранное` — сохранить кандидата;
- `Избранное` — показать избранных;
- `В чёрный список` — больше не показывать кандидата;
- `Новый поиск` — начать поиск заново;
- `Помощь` — показать команды.

---

## База данных

В проекте больше трёх таблиц:

1. `bot_users` — пользователи, которые общаются с ботом.
2. `candidates` — найденные кандидаты.
3. `photos` — фотографии кандидатов.
4. `favorites` — избранные кандидаты.
5. `blacklist` — чёрный список.
6. `viewed_candidates` — просмотренные кандидаты.

Схема базы находится в файле:

```text
db/schema.sql
```
